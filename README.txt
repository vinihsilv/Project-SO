Projeto acadêmico de Sistemas Operacionais. O projeto consiste em desenvolver uma simulação de troca de páginas utilizando a linguagem Python.

O projeto deve ser executado pela linha de comando, para execução é exigido 3 parâmetros, a quantidade de frames, a sequência de páginas e o nome em maiúsculo do algoritmo desejado (ex: FIFO,OPT,LRU)
